# Taklidy — Catalogue

An interactive two-page **book** catalogue of Taklidy fine silver-plate serveware,
crafted in Fes. Open `index.html` (or the published GitHub Pages site) to flip
through the collection with realistic page-turn spreads, jump-to-page, and an
EN/FR toggle.

## Contents
- `index.html` — the flipbook viewer (self-contained markup + styles)
- `js/page-flip.browser.js` — [StPageFlip](https://github.com/Nodlik/StPageFlip) (MIT), vendored for offline use
- `images/` — product photography
- `logo.png` — house monogram

No build step or server is required; it runs as static files.
